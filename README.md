# 道法自然

面向手机和桌面的纯静态传统术数网站。所有生辰、问题与排盘计算都保留在浏览器本地。

## 功能

- 四柱八字、五行十神、旺衰喜用、神煞
- 紫微斗数、大运与流年逐项详解
- 八字合婚与本次双方动态解释
- 黄历择吉
- 梅花易数：经典时间起卦、两数起卦、体用与互变分析
- 奇门遁甲：时家转盘、九宫、九星、八门、八神、天地盘与逐宫详解

## 本地运行

```bash
python3 -m http.server 8765 --bind 127.0.0.1
```

打开 `http://127.0.0.1:8765/`。

## 测试

```bash
node --test tests/core.test.mjs
```

测试包含梅花经典基准、报数边界、奇门权威盘基准及页面入口检查。

## 智能解释边界

当前详解采用本地、可追溯的规则知识层，不依赖网络模型，也不上传用户输入。若后续接入 DeepSeek、通义千问等模型，应仅通过服务器端接口调用，API 密钥不得写入前端文件；模型只能基于结构化排盘证据组织语言，不能改写排盘结果。
